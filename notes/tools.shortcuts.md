# Development Shortcuts

## Mac shortcuts
```
display desktop:                           Command (⌘) + Option (⌥) + D
Spotlight Search:                          Command (⌘) + space
switch app:                                Command (⌘) + Tab
switch same app different window:          Command (⌘) + ~
full screen mode:                          ⌃ + ⌘ + F
open mission control:                      ⌃ + ↑
```
