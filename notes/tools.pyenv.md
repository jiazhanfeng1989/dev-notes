# Pyenv Description
[pyenv](https://github.com/pyenv/pyenv) is a tool for managing multiple Python versions.


# Pyenv Commands
``` bash

# Install Python
pyenv install <version>

# Set local Python version
pyenv local <version>

# Set global Python version
pyenv global <version>

# List installed Python versions
pyenv versions

# Show the current Python version
pyenv version

# List available Python versions
pyenv install --list

# Uninstall Python
pyenv uninstall <version>

# Temporary set Python version for the current shell
pyenv shell <version>
```
